Internet via een omweg.

Soms worden er blokkades op het internet gezet. Bijvoorbeeld in Nederland is ThePirateBay, op rechtelijk bevel, geblokkeerd. Deze blokkade is lastig te omzeilen omdat de verbinding met de server geblokkeerd wordt in de server van jou provider. Toch is het mogelijk om toegang te krijgen tot een neutraal net (en dus toegang te krijgen tot de geblokkeerde server); met behulp van een Proxy Server.

Een proxy server is een soort tunnel. De computer maakt eerst verbinding met de server en verstuur een verzoek naar de server om een website op te vragen, de server vraagt dan de webpagina op en stuurt deze terug naar jouw computer. Doordat deze proxy server buiten Nederland staat wordt de website toegankelijk. Het is onmogelijk om een proxy server te blokkeren: vaak gaat het via standaard “internet wegen” waardoor het lijkt alsof je een normale website op vraag.

Een VPN, Virtual Private Netwerk, is een geavanceerde proxy server. Binnen een zo’n netwerk kunnen meerdere computers aan elkaar gekoppeld worden. Een VPN wordt vaak gebruikt om toegang te krijgen tot lokale servers binnen een bedrijf (van buiten af). Natuurlijk is het ook mogelijk om een VPN te gebruiken als proxy. Een VPN is meestal veiliger en sneller dan een proxy server.
